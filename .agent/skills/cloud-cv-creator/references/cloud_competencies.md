# Cloud Engineer & Architect Competencies

## Core Cloud Skills
- **Cloud Providers**: AWS (Amazon Web Services), Azure, Google Cloud Platform (GCP).
- **Infrastructure as Code (IaC)**: Terraform, AWS CloudFormation, Ansible, Pulumi.
- **Containerization & Orchestration**: Docker, Kubernetes (EKS, AKS, GKE), Helm.
- **CI/CD**: Jenkins, GitLab CI, GitHub Actions, AWS CodePipeline, Azure DevOps.
- **Scripting & Programming**: Python (Boto3), Go, Bash/Shell, JavaScript/TypeScript.
- **Networking**: VPC, Subnets, Route53, VPN, Direct Connect, Load Checksum (ALB/NLB).
- **Security**: IAM, KMS, WAF, Security Groups, Compliance (SOC2, HIPAA).
- **Monitoring & Describe**: Prometheus, Grafana, ELK Stack, CloudWatch, Datadog.

## Role-Specific Keywords
### Cloud Engineer
- Implementation, Deployment, Migration, Automation, Scripting, Troubleshooting, Maintenance.

### Cloud Architect
- Design, Strategy, Scalability, High Availability (HA), Disaster Recovery (DR), Cost Optimization, Governance, Best Practices.

## Example Bullet Points
- "Designed and implemented a serverless architecture using AWS Lambda and API Gateway, reducing infrastructure costs by 40%."
- "Automated infrastructure provisioning using Terraform and Jenkins pipelines, reducing deployment time from days to minutes."
- "Migrated a monolithic application to microservices on Kubernetes (EKS), improving scalability and fault tolerance."
- "Implemented a secure multi-account AWS landing zone strategy using Control Tower and AWS Organizations."
- "Optimized cloud spend by implementing auto-scaling policies and rightsizing instances, saving $50k annually."

## Certifications
- **AWS**: Certified Solutions Architect (Associate/Professional), Certified DevOps Engineer, Certified SysOps Administrator.
- **Azure**: Solutions Architect Expert, DevOps Engineer Expert, Administrator Associate.
- **GCP**: Professional Cloud Architect, Professional Cloud DevOps Engineer.
